## 2. 安装JDK

1.下载

* [JDK 17（windows, 解压版）]((https://download.oracle.com/java/17/latest/jdk-17_windows-x64_bin.zip))

2.解压

* C:\install\jdk-17.0.5

  ```txt
  C:\install\jdk-17.0.5
    bin
    conf
    include
    jmods
    legal
    lib
    LICENSE
    README
    release
  ```

3.验证

* 打开 CMD，执行 `C:\install\jdk-17.0.5\bin\java.exe -version`
